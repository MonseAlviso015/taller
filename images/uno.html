Que Es Un Sistema Informático. Definición Y Novedades

Con los avances tecnológicos que ha habido en los últimos 100 años con las computadoras, las personas cada vez quisieran saber más sobre ellas, una de las preguntas más frecuentes que se hacen los usuarios es de que es un sistema informático.

El sistema informático está compuesto por un (sub) sistema de hardware y un (sub) sistema de software. La primera es una combinación orgánica de varios componentes físicos basada en los principios de electricidad, magnetismo, luz y maquinaria, es la entidad en la que funciona el sistema.

Estos últimos son varios procedimientos y documentos utilizados para dirigir todo el sistema a trabajar de acuerdo con los requisitos especificados.

    Que es un sistema informático

    Desde la aparición de la primera computadora electrónica en 1946, la tecnología informática ha logrado un progreso sorprendente en componentes, estructuras de sistemas de hardware, sistemas de software y aplicaciones.

    Los sistemas informáticos modernos van desde microcomputadoras y computadoras personales hasta supercomputadoras y la red, con diversas formas y características, se ha utilizado ampliamente en informática científica, procesamiento de transacciones y control de procesos, y ha sobresalido cada vez más en diversos campos de la sociedad, y tiene un profundo impacto en el progreso social.

    Hay dos tipos de computadoras electrónicas: digitales y analógicas. En términos generales, la computadora se refiere a una computadora digital, y los datos procesados por su operación están representados por cantidades digitales discretas. Y la operación de procesamiento de datos de la computadora analógica está representada por un análogo continuo.

    En comparación con las máquinas digitales, las máquinas analógicas son rápidas, tienen interfaces simples con dispositivos físicos, pero tienen baja precisión, uso difícil, poca estabilidad y confiabilidad, y altos precios.

    Por lo tanto, el simulador se ha vuelto obsoleto y solo se usa en ocasiones que requieren una respuesta rápida pero poca precisión. La computadora híbrida que combina las ventajas de los dos tiene una cierta vitalidad.

    Características de un sistema informático

    El sistema informático se caracteriza por cálculos y juicios precisos y rápidos, buena versatilidad, fácil de usar y puede conectarse a una red.

    • Cálculo: las computadoras pueden realizar casi todos los cálculos complejos mediante operaciones aritméticas y lógicas.
    • Juicio: la computadora tiene la capacidad de distinguir diferentes situaciones y elegir diferentes procesamientos, por lo que puede usarse en administración, control, confrontación, toma de decisiones, razonamiento y otros campos.
    • Almacenamiento: las computadoras pueden almacenar grandes cantidades de información.
    • Más Rápido: el tiempo requerido para la operación de una computadora es tan pequeño como nanosegundos.
    • Fácil de usar: el rico software de alto rendimiento y la interfaz inteligente hombre-máquina lo facilitan enormemente.
    • Redes: múltiples sistemas informáticos pueden trascender los límites geográficos y compartir información remota y recursos de software con la ayuda de redes de comunicación.

    Composición

    El kernel es un sistema de hardware, un dispositivo físico real para el procesamiento de información. La capa más externa es la persona que usa la computadora, el usuario. La interfaz entre el sistema humano y el hardware es un sistema de software, que se puede dividir aproximadamente en tres capas: software del sistema, software de soporte y software de aplicación.

    Hardware

    El sistema de hardware se compone principalmente de una unidad central de procesamiento, memoria, sistema de control de entrada y salida y varios equipos externos. La unidad central de procesamiento es el componente principal para el procesamiento de información a alta velocidad, y su velocidad de procesamiento puede alcanzar cientos de millones de operaciones por segundo.

    Una memoria para almacenar programas, datos y archivos, a menudo causada por una memoria interna rápida (capacidad de hasta varios cientos de megabytes, o incluso la cantidad de bytes G) y una memoria externa lenta (capacidad de hasta decenas o cientos de G)composición.

    Varios dispositivos externos de entrada y salida son convertidores de información entre el hombre y la máquina, y el sistema de control de entrada-salida gestiona el intercambio de información entre los dispositivos externos y la memoria principal (unidad central de procesamiento).

    Software

    El software se divide en software del sistema, software de soporte y software de aplicación. El software del sistema se compone de sistema operativo, programa de utilidad, compilador, etc.

    El sistema operativo implementa la gestión y el control de diversos recursos de software y hardware. Los programas de utilidad están diseñados para la comodidad de los usuarios, como la edición de texto.

    La función del compilador es traducir el programa escrito por el usuario en lenguaje ensamblador o un cierto lenguaje de alto nivel en un programa de lenguaje máquina ejecutable por la máquina.

    Firmware

    El firmware es la programación que se escribe en la memoria no volátil de un dispositivo de hardware. La memoria no volátil es una forma de memoria estática de acceso aleatorio cuyo contenido se guarda cuando un dispositivo de hardware se apaga o pierde su fuente de alimentación externa.

    El firmware se añade en el momento de la fabricación, se utiliza para ejecutar programas de usuario en el dispositivo y puede considerarse como el software que permite ejecutar el hardware.

    Los fabricantes de hardware utilizan el firmware incorporado para controlar las funciones de varios dispositivos y sistemas de hardware, de manera muy similar a como el sistema operativo (SO) de una computadora controla la función de las aplicaciones de software.

    El firmware puede escribirse en la memoria de solo lectura (ROM), en la memoria de solo lectura programable y borrable (EPROM) o en la memoria flash. El firmware que está incorporado en los chips de memoria flash puede actualizarse más fácilmente que el firmware escrito en la ROM o EPROM, lo que lo hace más adaptable.

    Avances de un sistema informático

    El sistema informático se actualiza aproximadamente cada 3 a 5 años, la relación rendimiento-precio se multiplica por diez y el volumen se reduce considerablemente. La tecnología de circuito integrado a muy gran escala continuará desarrollándose rápidamente y tendrá un impacto enorme y profundo en varios sistemas informáticos.

    Han aparecido microcomputadoras de 32 bits, y también han salido microcomputadoras de 64 bits. No está lejos hacer 10 millones de componentes en un solo chip.

    La investigación en dispositivos que son 10 a 100 veces más rápidos que los circuitos integrados de, como arseniuro de galio, dispositivos de alta movilidad de electrones, uniones josephson y componentes ópticos, tendrá resultados importantes.

    La tecnología de microensamblaje para aumentar la densidad de ensamblaje y acortar la línea de interconexión es una de las tecnologías clave de la nueva generación de computadoras. La comunicación por fibra óptica será ampliamente utilizada.

    Varios dispositivos periféricos inteligentes de alta velocidad emergen constantemente, y la llegada de los discos ópticos hará que el almacenamiento masivo auxiliar sea completamente nuevo. Los sistemas de múltiples computadoras y los sistemas de procesamiento distribuido serán estructuras de sistema llamativos.

    Fortalecimiento del software

    El fortalecimiento del software (llamado firmware) es una tendencia de desarrollo. Se han comenzado a utilizar en la práctica nuevos tipos de máquinas que no son de Neumann, computadoras de razonamiento, computadoras de base de conocimiento, etc. El desarrollo de software eliminará el atraso y la ineficiencia. La está desarrollando en profundidad.

    La producción de software se está desarrollando en la dirección de ingeniería, formalización, automatización, modularización e integración. La investigación de nuevos lenguajes de alto nivel, como lenguajes lógicos, lenguajes funcionales e inteligencia artificial, hará que las interfaces hombre-máquina sean simples y naturales (puede ver, escuchar, hablar y dibujar directamente). La tecnología de bases de datos se desarrollará en gran medida. Estarán muy extendidas.

    Los sistemas informáticos de próxima generación se caracterizan por una gran potencia de procesamiento (por ejemplo, 10 a 100 mil millones de operaciones por segundo), una gran base de conocimiento e información y una alta inteligencia se están desarrollando vigorosamente. Las aplicaciones informáticas se extenderán cada vez más.
